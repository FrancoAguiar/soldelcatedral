# Sol del Catedral — sitio Next.js

## Ejecutar localmente

```bash
npm install
npm run dev
```

Abrí http://localhost:3000

## Publicar

1. Subí esta carpeta a GitHub.
2. Importá el repositorio en Vercel.
3. Vercel detectará Next.js y publicará el sitio.

## Antes de publicar

- Reemplazá `59800000000` por el WhatsApp real en `app/page.tsx` y `components/Nav.tsx`.
- Cambiá textos, capacidad y servicios por los datos reales.
- Reemplazá enlaces del footer.
